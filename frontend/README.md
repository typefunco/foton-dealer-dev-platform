# FOTON Dealer Development Platform - Frontend

## Описание проекта

Платформа для развития дилеров FOTON с комплексной аналитикой производительности, включающая:

- **General Info**: Информация о дилере, регионе, городе, менеджере
- **Dealer Development**: Чек-лист, класс дилерства, брендинг, маркетинговые инвестиции, сопутствующие бизнесы
- **Sales**: Стоки HDT/MDT/LDT, выкуп, целевые продажи, команда продаж, тренинги, контракты
- **AfterSales**: Рекомендуемые стоки, гарантийные стоки, часы работы Foton, контракты, тренинги, CSI
- **Performance**: Выручка, прибыль, маржа, позиция Foton, совместные рекомендации

## Внесенные изменения согласно требованиям заказчика

### 1. Обновленная логика домашней страницы
- **Правильная последовательность выбора параметров:**
  1. Region (All, Central, North-West, Volga, South, N. Caucasus, Ural, Siberia, Far East)
  2. Dealers (All или выбранные дилеры региона)
  3. Parameters (All, Total Performance, Dealer Development, Sales, AfterSales)
  4. Period From (QX 2025)
  5. Period To (QX 2025)
  6. Find Results

### 2. Исправление названий разделов
- Заменено "Auto Sales" на "AfterSales" во всех компонентах
- Обновлены интерфейсы и типы данных

### 3. Новые маркетинговые показатели
- Добавлены поля для брендинга (Y/N)
- Добавлены маркетинговые инвестиции в рублях за квартал
- Подготовлена структура для сопутствующих бизнесов дилеров

### 4. Интеграция логотипов брендов
- Создан компонент `BrandLogos` для отображения логотипов
- Поддержка 16 брендов: Shacman, Dongfeng, Sitrak, FAW, JAC, Isuzu, Kamaz, MAZ, GAZ, Valdai, Foton, SDAC, Ambertruck, Sollers, Chenglong, Sany
- Tooltip с названием бренда при наведении
- Fallback для отсутствующих логотипов

### 5. Переработка графиков на странице дилера
- **Sales Target**: Заменен на pie chart с выполнением плана по году и кварталу
- **Sales Performance**: Убран Sales Target, оставлены Sales Revenue и AfterSales
- **Цвета стоков**: Единые цвета для HDT, MDT, LDT в стоках и выкупе

## Технологии

- **React 18** с TypeScript
- **Tailwind CSS** для стилизации
- **Recharts** для графиков и диаграмм
- **React Router** для навигации
- **Vite** для сборки

## Установка и запуск

### Предварительные требования
- Node.js 18+ 
- npm или yarn

### Установка зависимостей
```bash
npm install
# или
yarn install
```

### Запуск в режиме разработки
```bash
npm run dev
# или
yarn dev
```

Приложение будет доступно по адресу: http://localhost:5173

### Сборка для продакшена
```bash
npm run build
# или
yarn build
```
